## About The Project
Tab Hopper Link: https://chromewebstore.google.com/detail/tab-hopper/jjkakfndajglibanfbjnjeeicdbmbpcl?authuser=6&hl=en-GB

Tab Hopper is a browser extension tool that can make it easier to quickly switch and close opened tabs. Having a bunch of tabs opened can make it different to navigate to the one you want. By using Tab Hopper, you can visually see all youre opened tabs.


### Built With
Javascript and 
HTML/CSS


<!-- GETTING STARTED -->
## Getting Started
Click the link to add the chrome extension to your browser.
After that, it is important to refresh all existing opened tabs so that the chrome extension can inject the content script. Note that chrome does not allow content script injecting into it internal pages so you cannot use this extension on those pages.
